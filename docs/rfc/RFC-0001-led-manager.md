## Objetivo

Criar um módulo responsável por controlar toda a comunicação visual do satélite.

## Motivação

Centralizar o controle do LED RGB, evitando que outros módulos manipulem diretamente o hardware.

## Responsabilidades

- Estado do sistema
- Efeitos luminosos
- Diagnóstico
- Feedback visual
- OTA
- Inicialização

## Benefícios

- Código organizado
- Fácil manutenção
- Reutilização
