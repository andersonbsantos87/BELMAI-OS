# ADR-001

## Decisão

Utilizar ESP-IDF como framework padrão do BELMAI OS.

## Motivo

Maior estabilidade para aplicações de áudio e melhor compatibilidade com os componentes de Voice Assistant do ESPHome.

## Consequências

Todos os satélites BELMAI utilizarão ESP-IDF como padrão, salvo necessidade técnica específica.
